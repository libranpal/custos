About OAuth 2.1

OAuth 2.1 is the latest version of the OAuth protocol, which provides secure delegated access to resources. This demo implements:

  • Authorization Code Flow with PKCE
  • Client Credentials Flow
  • Refresh Token Rotation
  • Token Revocation
  • Token Introspection
